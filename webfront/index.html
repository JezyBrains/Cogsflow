Integration with CodeIgniter

To integrate these pages with your CodeIgniter application:

  1. Copy the CSS files to public/webfront-assets/
  2. Copy the images to public/webfront-images/
  3. Copy the JavaScript to public/webfront-assets/
  4. Update your CodeIgniter views to use the HTML structure
  5. Update image paths to use base_url()
